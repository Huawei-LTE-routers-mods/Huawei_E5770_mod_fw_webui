Untuk menyediakan layanan yang lebih baik, perangkat ini akan secara otomatis mengambil informasi pembaruan perangkat lunak dari Huawei atau operator Anda setelah menghubungkan ke Internet. Proses ini akan menggunakan data seluler, juga memerlukan akses ke pengenal unik perangkat Anda (IMEI/SN) dan ID jaringan penyedia layanan (PLMN) untuk memeriksa apakah perangkat Anda perlu diperbarui. Selain itu, perangkat ini mendukung pengunduhan dan pemasangan pembaruan penting dari Huawei dalam situasi darurat. Fungsi ini diaktifkan berdasarkan pengaturan standar. Untuk menonaktifkan fungsi ini, harap masuk ke situs web pengelolaan produk dan melakukan perubahan yang diperlukan di menu pengaturan.
